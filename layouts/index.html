01. About

Future Gadget Labs is a personal engineering lab focused on building data pipelines, admin tooling, and analytics platforms for niche domains that deserve better software. Every project starts with a real data problem and ends with a system that makes it invisible.

All projects share a common backbone — static frontends served from GitHub Pages, writes handled by authenticated Cloud Run APIs, and everything persisted in BigQuery. The pattern is boring on purpose: consistent infrastructure means the interesting work stays in the domain.
