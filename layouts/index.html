⚡
Read/Write Separation

Reads are served from pre-built static JSON (GitHub Raw or GCS), keeping the UI fast and API load minimal. Writes go through authenticated Cloud Run endpoints.

🔒
Defense in Depth Auth

Firebase Auth with a server-side email allowlist enforced at both the frontend and the Cloud Run API layer using ID token validation.

⚙
Zero-Config CI/CD

GitHub Actions handles build and deploy to GitHub Pages on every push. Secrets and config live as HUGO_PARAMS_* environment variables in the Actions workflow.
